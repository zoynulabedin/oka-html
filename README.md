# One Kingdom Church Website

A modern, animated church website built with HTML, Tailwind CSS, and JavaScript.

## Features

- **3 Pages**: Home, About, and Sundays
- **Smooth Animations**: Fade-ins, slides, parallax effects, and Ken Burns effect on hero images
- **Responsive Design**: Mobile-friendly layout
- **Interactive Elements**:
  - FAQ accordion
  - Hover effects on cards and buttons
  - Animated navigation
  - Scrolling marquee
  - Floating logo animations
- **Modern Design**:
  - Glassmorphism effects
  - Gradient backgrounds
  - Custom color palette (Navy Blue #0A3B5C, Crimson Red #C41E3A, Light Blue #B8D4E8)
  - Google Fonts (Poppins & Playfair Display)

## Pages

### Home (index.html)

- Hero section with animated logo
- Scrolling marquee
- Location card with map
- One Kingdom section
- Three feature sections
- The Salt Company section
- Footer

### About (about.html)

- Hero section with OKC badge
- Mission statement
- Our Story section
- G.R.A.C.E. values breakdown
- Core Beliefs section
- Leadership section

### Sundays (sundays.html)

- Hero section
- Location card
- Welcome message
- FAQ accordion

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Google Fonts
- CSS Animations

## How to Run

Simply open `index.html` in your web browser. No build process required!

## Color Palette

- Navy Blue: `#0A3B5C`
- Crimson Red: `#C41E3A`
- Light Blue: `#B8D4E8`
- Beige: `#F5E6D3`

## Animations

- Ken Burns effect on hero backgrounds
- Fade-in animations on scroll
- Slide animations (left, right, down)
- Floating logo animation
- Slow spinning badge
- Parallax scrolling
- Marquee text scrolling
- Hover scale effects

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
